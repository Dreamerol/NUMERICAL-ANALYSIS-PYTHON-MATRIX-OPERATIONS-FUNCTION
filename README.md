## 🚀 Overview
This repository contains solutions and examples from the **Numerical Analysis** course at FMI. It focuses on **numerical methods** such as polynomial interpolation, approximation, and fitting data points using classical algorithms.

The work is implemented primarily in Jupyter Notebooks and demonstrates how numerical techniques can be used to model and analyze mathematical data.

---

## 📌 What Is Numerical Analysis?

Numerical analysis is the study and implementation of algorithms that provide numerical approximations to mathematical problems that are often impossible or impractical to solve symbolically. It includes methods for approximating functions, solving nonlinear equations, interpolating data points, and more.

---

## 🧠 Key Concepts Covered

### 📈 Polynomial Interpolation

Interpolation is a method of estimating values between a discrete set of known data points. In this project, **polynomial interpolation** is used to fit a polynomial that passes through a given set of points `(x_i, y_i)` such that we can approximate values at new positions. :contentReference[oaicite:1]{index=1}

There are many ways to construct such polynomials, including:

- **Lagrange Interpolation** — builds the interpolating polynomial directly from the data points.
- **Newton’s Divided Differences** — a form that adapts easily when adding new points.

Both methods result in a polynomial that fits the given data.
